## Exploratory Data Analysis
### Course Projects

This repository contains the code, plots and data for the "Exploratory Data Analysis" Coursera Course.

To run the code for a Project:
  1. Download the the required Project folder as is to a folder of your choice on the your computer
  2. Unzip the file inside the Data folder to reside directly under the Data folder
  3. Go to or change your directory in R studio to Code under the Project folder.
  4. Run each R file seperately: plot1.R, plot2.R, plot3.R, ...
  5. The plots will be created under the Plots as plot1.png, plot2.png, plot3.png, ...
